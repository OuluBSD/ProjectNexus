# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0-rc1] - 2025-12-07

### Added
- Full command tree across all namespaces
- Streaming observability (UOL)
- AI chat sessions with token streaming
- Network & debug inspection tools
- Pretty output mode + hints system
- System doctor command
- Man pages + shell completions
- Installer scripts for all platforms
- Version command with git metadata
- Release & build pipeline
- RC1 certification validator

### Changed
- All error outputs standardized
- Exit codes unified for automation
- Manifest-driven help system

### Fixed
- TypeScript errors & nullability issues
- ESM loader incompatibilities in tools
- Handler registry inconsistencies

### Security
- Auth token stored in user config
- Dependency audit included in RC1 gates
